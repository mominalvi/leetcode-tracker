# Project Instructions

This is a very small personal Python CLI tool.

Goals:
- Keep it simple and practical
- No overengineering
- Beginner-intermediate friendly code
- Minimal file structure
- Clean CLI UX
- Not a GUI
- Not a fancy TUI unless clearly justified

Constraints:
- Python only
- Standard library only unless absolutely necessary
- JSON storage
- CLI only
- Prefer readability over cleverness
- Prefer simple functions over unnecessary abstractions
- Stay aligned with a 1-2 day build

Working style:
- Read implementation_spec.md first
- Follow it closely
- Ask before adding features outside the spec
- Suggest clean commit boundaries before committing
